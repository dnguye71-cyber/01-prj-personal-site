# Dylan Nguyen's Personal Portfolio

This is a modern and responsive portfolio website showcasing Dylan's projects, skills and experience as a CSE student passionate about learning new skills on technologies and showcasing his skills and experience in web development.

## Project Overview

This portfolio serves as a comprehensive showcase of his abilities in web development, API interaction, game development, and modern Javascript practices.

## Features

### Responsive Design
- **Mobile-First Approach**: Fully optimized for mobile, tablet, and desktop devices
- **Flexible Grid Layouts**: Uses CSS Grid and Flexbox for adaptive layouts
- **Smooth Scrolling**: Enhanced user experience with smooth navigation between sections

### Modern Styling
- **Gradient Backgrounds**: Professional color scheme with purple/blue gradients
- **Interactive Elements**: Hover effects on project cards, skill badges, and navigation links
- **Accessibility-Focused**: Semantic HTML structure and proper contrast ratios
